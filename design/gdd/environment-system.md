## Overview

**环境系统**是战斗系统的核心组件，负责管理环境/地形对战斗的影响。它定义了不同环境类型（如森林、沙漠、雪山、火山等）及其对卡牌效果、召唤物属性、元素克制等的影响。

玩家通过环境系统感受到"战斗环境在不断变化，我的策略需要适应"——每个环境都有独特的规则和效果，玩家需要根据环境调整卡组和战术。

## Player Fantasy

**环境系统**是玩家直接体验的系统——玩家在战斗中感受到环境的影响。

**情感目标**："战斗环境在不断变化，我的策略需要适应"

**玩家应该感受到**：
- **策略深度**：我需要根据环境调整卡组和战术
- **探索发现**：每个环境都有独特的规则和效果
- **掌控感**：我可以通过选择环境来影响战斗结果

**情感锚点**：像杀戮尖塔中不同楼层有不同的环境效果——"这个楼层是火焰环境，我需要调整策略！"

## Detailed Design

### Core Rules

1. **环境类型**：
   - **森林环境**：增加土元素伤害，减少火元素伤害
   - **沙漠环境**：增加火元素伤害，减少水元素伤害
   - **雪山环境**：增加水元素伤害，减少火元素伤害
   - **火山环境**：增加火元素伤害，减少土元素伤害
   - **海洋环境**：增加水元素伤害，减少土元素伤害
   - **天空环境**：增加风元素伤害，减少土元素伤害

2. **环境效果**：
   - **元素加成**：环境增加特定元素伤害25%
   - **元素削弱**：环境减少特定元素伤害25%
   - **召唤物影响**：环境影响召唤物属性（如森林环境增加土元素召唤物生命值）
   - **卡牌效果**：环境影响卡牌效果（如火山环境增加火焰卡牌伤害）

3. **环境切换**：
   - **战斗开始**：战斗开始时随机选择环境
   - **环境卡牌**：玩家可以使用环境卡牌切换环境
   - **环境持续**：环境效果持续到战斗结束或被切换

4. **环境限制**：
   - **每场战斗只能切换一次环境**
   - **环境效果不能叠加**
   - **环境效果不能被免疫**

### States and Transitions

| 状态 | 描述 | 触发条件 |
|------|------|----------|
| **环境选择** | 战斗开始时选择环境 | 战斗开始时进入 |
| **环境激活** | 环境效果生效中 | 环境选择后进入 |
| **环境切换** | 切换到新环境 | 使用环境卡牌时进入 |
| **环境结束** | 环境效果结束 | 战斗结束或环境被切换 |

**状态转换规则**：
- 环境选择 → 环境激活：环境选择后自动进入
- 环境激活 → 环境切换：使用环境卡牌时进入
- 环境切换 → 环境激活：新环境生效后进入
- 环境激活 → 环境结束：战斗结束或环境被切换

### Interactions with Other Systems

| 系统 | 数据流向 | 接口描述 |
|------|----------|----------|
| **卡牌数据库** | 输入 | 读取环境卡牌定义（环境类型、效果） |
| **伤害计算系统** | 输出 | 提供环境元素加成/削弱值 |
| **召唤物系统** | 输出 | 提供环境对召唤物属性的影响 |
| **卡牌战斗系统** | 输出 | 提供当前环境状态和切换规则 |
| **UI系统** | 输出 | 提供环境显示和切换提示 |

## Formulas

### 环境元素加成公式

`environment_element_bonus = base_bonus × environment_level`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_bonus | E_b | float | 0.25 | 环境基础元素加成 |
| environment_level | E_l | int | 1-3 | 环境等级 |
| environment_element_bonus | E_e | float | 0.25-0.75 | 环境元素加成值 |

**输出范围：** 0.25到0.75（正常游戏范围）
**示例：** 环境等级2，元素加成 = 0.25 × 2 = 0.50

### 环境召唤物影响公式

`environment_summon_bonus = base_summon_bonus × environment_level`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_summon_bonus | S_b | float | 0.10 | 环境基础召唤物加成 |
| environment_level | E_l | int | 1-3 | 环境等级 |
| environment_summon_bonus | S_e | float | 0.10-0.30 | 环境召唤物加成值 |

**输出范围：** 0.10到0.30（正常游戏范围）
**示例：** 环境等级2，召唤物加成 = 0.10 × 2 = 0.20

### 环境卡牌效果公式

`environment_card_bonus = base_card_bonus × environment_level`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_card_bonus | C_b | float | 0.15 | 环境基础卡牌效果加成 |
| environment_level | E_l | int | 1-3 | 环境等级 |
| environment_card_bonus | C_e | float | 0.15-0.45 | 环境卡牌效果加成值 |

**输出范围：** 0.15到0.45（正常游戏范围）
**示例：** 环境等级2，卡牌效果加成 = 0.15 × 2 = 0.30

## Edge Cases

- **如果环境切换时战斗已结束**：环境切换不执行，显示"战斗结束"提示。
- **如果环境切换时已切换过一次**：环境切换不执行，显示"已切换过环境"提示。
- **如果环境效果叠加**：环境效果不叠加，取最高值。
- **如果环境效果被免疫**：环境效果正常执行，不能被免疫。
- **如果环境效果与卡牌效果冲突**：环境效果优先。
- **如果环境效果与召唤物效果冲突**：环境效果优先。
- **如果环境效果与状态效果冲突**：环境效果优先。
- **如果环境效果与元素克制冲突**：环境效果优先。

| **环境持续时间** | 整场战斗 | — | 环境效果持续时间 |

## Visual/Audio Requirements

### 视觉效果
- **环境背景**：战斗背景根据环境类型变化（森林/沙漠/雪山/火山/海洋/天空）
- **环境粒子效果**：环境类型对应的粒子效果（树叶/沙尘/雪花/火焰/水滴/云朵）
- **环境切换动画**：环境切换时，背景平滑过渡，粒子效果变化
- **环境效果提示**：环境效果生效时，显示环境图标和效果说明
- **环境等级显示**：环境等级用星级显示（1-3星）

### 音频效果
- **环境背景音乐**：环境类型对应的背景音乐（森林/沙漠/雪山/火山/海洋/天空）
- **环境音效**：环境类型对应的环境音效（风声/沙沙声/雷声/火焰声/海浪声/鸟鸣声）
- **环境切换音效**：环境切换时，播放切换音效
- **环境效果音效**：环境效果生效时，播放效果音效

## UI Requirements

### 环境状态显示
- **位置**：战斗界面左上角
- **内容**：当前环境类型、环境等级、环境效果
- **大小**：200x100像素
- **动画**：环境切换时，环境图标跳动

### 环境切换提示
- **位置**：屏幕中央偏上
- **内容**：环境名称、环境效果
- **动画**：从中心向上漂浮，2秒后消失
- **颜色**：环境类型对应颜色（森林绿色/沙漠黄色/雪山白色/火山红色/海洋蓝色/天空青色）

### 环境效果详情
- **显示内容**：环境名称、环境效果、环境等级
- **布局**：垂直列表，按环境类型分组
- **交互**：悬停显示环境效果详情
