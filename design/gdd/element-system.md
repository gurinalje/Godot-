## Overview

**元素系统**是游戏的核心系统之一，负责管理元素切换和卡牌池解锁。它定义了元素的类型、克制关系、切换规则和解锁条件，为卡牌数据库、卡牌战斗系统、召唤物系统和环境系统提供元素支持。玩家通过元素系统感受到"我选择的元素影响了我的卡牌和战斗策略"——元素切换不仅改变了可用卡牌池，还影响了战斗中的元素克制关系。

元素系统读取选择系统的元素选择数据，结合卡牌数据库的元素卡牌定义，实现元素切换和卡牌池解锁。玩家通过元素系统感受到"我的元素选择决定了我的战斗风格，我需要根据敌人选择合适的元素"。

## Player Fantasy

**元素系统**是玩家直接体验的系统——玩家通过选择元素，影响卡牌池和战斗策略。

**情感目标**："我选择的元素影响了我的卡牌和战斗策略"

**玩家应该感受到**：
- **掌控感**：我选择的元素决定了我的卡牌池和战斗风格
- **策略深度**：我需要根据敌人选择合适的元素
- **惊喜感**：发现新的元素组合和克制关系

**情感锚点**：像宝可梦中选择属性的策略感——"我选择了火属性，但对手是水属性，我需要调整策略！"

## Detailed Design

### Core Rules

1. **元素类型**：
   - **火（Fire）**：攻击型元素，克制风，被水克制
   - **水（Water）**：防御型元素，克制火，被土克制
   - **风（Wind）**：速度型元素，克制土，被火克制
   - **土（Earth）**：生命型元素，克制水，被风克制
   - **雷（Thunder）**：特殊元素，对所有元素有加成，独立于循环克制

2. **元素克制关系**：
   - 火 > 风 > 土 > 水 > 火（循环克制）
   - 雷独立，对所有元素有1.2x加成
   - 克制倍率：1.2x（攻击方优势）
   - 被克制倍率：0.8x（攻击方劣势）

3. **元素切换规则**：
   - 玩家可以在探索界面切换元素
   - 切换元素后，卡牌池自动更新
   - 切换元素有冷却时间（默认3回合）
   - 切换元素时，当前卡牌保留，但新卡牌池解锁

4. **元素卡牌池解锁**：
   - 每个元素有独特的卡牌池
   - 解锁元素后，该元素卡牌池自动解锁
   - 卡牌池中的卡牌可以通过战斗获得
   - 卡牌池中的卡牌可以通过商店购买

5. **元素加成**：
   - 玩家选择的元素对应该元素的卡牌有加成
   - 元素加成倍率：1.1x（默认）
   - 元素加成可以通过印记系统增强

### States and Transitions

| 状态 | 描述 | 触发条件 |
|------|------|----------|
| **无元素** | 玩家未选择元素 | 游戏开始时进入 |
| **火元素** | 玩家选择火元素 | 选择火元素时进入 |
| **水元素** | 玩家选择水元素 | 选择水元素时进入 |
| **风元素** | 玩家选择风元素 | 选择风元素时进入 |
| **土元素** | 玩家选择土元素 | 选择土元素时进入 |
| **雷元素** | 玩家选择雷元素 | 选择雷元素时进入 |

**状态转换规则**：
- 无元素 → 任意元素：选择元素时进入
- 任意元素 → 任意元素：切换元素时进入（冷却时间结束后）
- 任意元素 → 无元素：重置元素时进入

## Formulas

### 元素克制倍率公式

`element_modifier = base_modifier × element_advantage`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_modifier | M_b | float | 1.0 | 基础倍率（固定值） |
| element_advantage | E_a | float | 0.8-1.2 | 元素克制/被克制倍率 |
| element_modifier | M_e | float | 0.8-1.2 | 最终元素倍率 |

**输出范围：** 0.8到1.2（正常游戏范围）
**示例：** 火攻击风，倍率 = 1.0 × 1.2 = 1.2x

### 元素加成公式

`element_bonus = base_bonus × (1 + story_mark_bonus)`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_bonus | B_b | float | 0.1 | 基础元素加成（默认10%） |
| story_mark_bonus | S_b | float | 0-0.5 | 故事印记加成（来自印记系统） |
| element_bonus | B_e | float | 0.1-0.15 | 最终元素加成 |

**输出范围：** 0.1到0.15（正常游戏范围）
**示例：** 基础加成10%，印记加成20%，加成 = 0.1 × 1.2 = 0.12（12%）

### 元素切换冷却公式

`cooldown_remaining = max(0, last_switch_turn + cooldown_duration - current_turn)`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| last_switch_turn | L_s | int | 0-1000 | 上次切换回合 |
| cooldown_duration | C_d | int | 3 | 冷却持续时间（回合） |
| current_turn | T_c | int | 0-1000 | 当前回合 |
| cooldown_remaining | C_r | int | 0-3 | 剩余冷却时间 |

**输出范围：** 0到3（正常游戏范围）
**示例：** 上次切换回合5，当前回合7，冷却 = max(0, 5+3-7) = 1回合

## Edge Cases

- **如果玩家尝试切换元素但冷却时间未结束**：显示"冷却中"提示，无法切换。
- **如果玩家尝试选择已选择的元素**：显示"已选择该元素"提示，无法重复选择。
- **如果玩家尝试在战斗中切换元素**：显示"战斗中无法切换"提示，无法切换。
- **如果玩家尝试在对话中切换元素**：显示"对话中无法切换"提示，无法切换。
- **如果玩家尝试切换到未解锁的元素**：显示"元素未解锁"提示，无法切换。
- **如果玩家尝试切换元素但卡牌池为空**：显示"卡牌池为空"提示，无法切换。
- **如果玩家尝试切换元素但卡牌正在使用中**：显示"卡牌使用中"提示，无法切换。
- **如果玩家尝试切换元素但游戏暂停**：显示"游戏暂停"提示，无法切换。

## Tuning Knobs

| 调优参数 | 默认值 | 范围 | 说明 |
|----------|--------|------|------|
| **元素克制倍率** | 1.2x | 1.1x-1.5x | 元素克制时的伤害加成 |
| **元素被克制倍率** | 0.8x | 0.5x-0.9x | 元素被克制时的伤害减免 |
| **元素加成倍率** | 1.1x | 1.0x-1.3x | 元素对应卡牌的伤害加成 |
| **元素切换冷却** | 3回合 | 1-5回合 | 切换元素后的冷却时间 |
| **元素卡牌池大小** | 20张 | 10-30张 | 每个元素的卡牌池大小 |
| **元素解锁条件** | 1个印记 | 1-3个印记 | 解锁元素需要的印记数量 |
| **元素加成印记系数** | 0.5 | 0.3-0.7 | 印记对元素加成的影响系数 |
| **雷元素独立加成** | 1.2x | 1.1x-1.5x | 雷元素对所有元素的加成 |

## Visual/Audio Requirements

### 视觉效果
- **元素图标**：每个元素有独特的图标（火焰/水滴/风暴/岩石/雷电）
- **元素颜色**：每个元素有独特的颜色（火红/水蓝/风绿/土黄/雷紫）
- **元素切换动画**：切换元素时，播放元素转换特效
- **元素克制提示**：元素克制时，显示克制图标（向上箭头）
- **元素被克制提示**：元素被克制时，显示被克制图标（向下箭头）
- **元素加成提示**：元素加成时，显示加成图标（星星）

### 音频效果
- **元素切换音效**：切换元素时，播放元素转换音效（根据元素类型变化）
- **元素克制音效**：元素克制时，播放克制音效（强化音效）
- **元素被克制音效**：元素被克制时，播放被克制音效（弱化音效）
- **元素加成音效**：元素加成时，播放加成音效（增益音效）
- **元素解锁音效**：解锁新元素时，播放解锁音效（成就音效）

## UI Requirements

### 元素选择界面
- **位置**：探索界面右下角
- **内容**：5个元素图标（火/水/风/土/雷）
- **状态**：已选择元素高亮显示，未解锁元素灰色显示
- **交互**：点击元素图标切换元素

### 元素状态显示
- **位置**：战斗界面左上角，玩家信息区域
- **内容**：当前元素图标、元素名称、元素加成倍率
- **大小**：50x50像素
- **动画**：元素切换时，图标闪烁

### 元素克制提示
- **位置**：战斗界面中央，敌人信息区域
- **内容**：元素克制/被克制图标
- **大小**：30x30像素
- **动画**：元素克制时，图标跳动

### 元素冷却显示
- **位置**：元素选择界面，元素图标下方
- **内容**：冷却剩余回合数
- **大小**：20x20像素
- **动画**：冷却中时，数字倒计时
