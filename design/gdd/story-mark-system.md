# 故事印记系统 (Story Mark System)

> **Status**: In Design  
> **Author**: [user + agents]

> **Implements Pillar**: 支柱1 — 选择即规则

## Overview

**故事印记系统**是游戏的核心创新系统，负责管理玩家在游戏过程中做出的选择所留下的"印记"。这些印记会影响卡牌效果、战斗规则和游戏世界。玩家通过故事印记系统感受到"我的选择真的改变了游戏"——每一个决定都会在卡牌上留下印记，改变卡牌的行为。

故事印记系统实现了游戏的核心支柱"选择即规则"——玩家的选择不仅改变故事，还改变卡牌的效果和战斗规则。玩家通过故事印记系统感受到"我的选择产生了有意义的结果，我的卡牌因为我之前的选择而变得更强/更弱"。

## Player Fantasy

**故事印记系统**是玩家直接体验的系统——玩家通过做出选择来留下印记，印记会影响卡牌效果和战斗规则。

**情感目标**："我的选择真的改变了游戏"

**玩家应该感受到**：
- **掌控感**：我的选择直接影响卡牌效果和战斗规则
- **策略深度**：我需要考虑选择的长期影响，而不仅仅是眼前利益
- **惊喜感**：发现印记如何改变卡牌效果时的惊喜

**情感锚点**：像传说之下的选择系统——"我救了这个人，现在我的卡牌获得了治疗效果"

## Detailed Design

### Core Rules

1. **印记类型**：
   - **善良印记**：帮助NPC、做出善良选择时获得
   - **邪恶印记**：伤害NPC、做出邪恶选择时获得
   - **中立印记**：做出中立选择时获得
   - **隐藏印记**：完成隐藏任务、发现秘密时获得

2. **印记效果**：
   - **卡牌效果改变**：印记会改变卡牌的效果（如善良印记使治疗卡牌效果+20%）
   - **战斗规则改变**：印记会改变战斗规则（如邪恶印记使所有伤害+10%）
   - **解锁新卡牌**：印记可以解锁新卡牌（如善良印记解锁"圣光"卡牌）
   - **改变NPC态度**：印记会改变NPC对玩家的态度（如善良印记使NPC更友好）

3. **印记叠加规则**：
   - **同类印记**：效果叠加（如两个善良印记，治疗效果+40%）
   - **异类印记**：效果冲突（如善良印记和邪恶印记同时存在，效果相互抵消）
   - **印记上限**：每个类型最多3个印记

4. **印记获取时机**：
   - **对话选择**：在对话中做出选择时获得
   - **任务完成**：完成任务时获得
   - **探索发现**：发现隐藏内容时获得
   - **战斗胜利**：战斗胜利时获得（根据战斗方式）

5. **印记存储**：
   - **持久化**：印记永久保存，不会因为游戏结束而消失
   - **跨存档**：印记在所有存档中共享
   - **可查看**：玩家可以在游戏菜单中查看自己的印记

### States and Transitions

| 状态 | 描述 | 触发条件 |
|------|------|----------|
| **无印记** | 玩家没有印记 | 游戏开始时 |
| **善良印记** | 玩家拥有善良印记 | 做出善良选择时获得 |
| **邪恶印记** | 玩家拥有邪恶印记 | 做出邪恶选择时获得 |
| **中立印记** | 玩家拥有中立印记 | 做出中立选择时获得 |
| **隐藏印记** | 玩家拥有隐藏印记 | 完成隐藏任务时获得 |

**状态转换规则**：
- 无印记 → 善良印记：做出善良选择时自动进入
- 无印记 → 邪恶印记：做出邪恶选择时自动进入
- 无印记 → 中立印记：做出中立选择时自动进入
- 无印记 → 隐藏印记：完成隐藏任务时自动进入
- 善良印记 → 善良印记：做出善良选择时，印记效果叠加
- 邪恶印记 → 邪恶印记：做出邪恶选择时，印记效果叠加

## Formulas

### 印记效果公式

`mark_effect = base_effect × (1 + mark_count × mark_multiplier)`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_effect | E_b | float | 0-1.0 | 卡牌基础效果（如治疗量、伤害量） |
| mark_count | M_c | int | 1-3 | 同类印记数量 |
| mark_multiplier | M_m | float | 0.1-0.3 | 每个印记的加成倍率 |
| mark_effect | E_m | float | 0-2.0 | 印记影响后的最终效果 |

**输出范围：** 0到2.0（正常游戏范围）
**示例：** 基础治疗效果0.5，2个善良印记，加成倍率0.2，效果 = 0.5 × (1 + 2 × 0.2) = 0.7

### 印记冲突公式

`conflict_resolution = good_marks - evil_marks`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| good_marks | G_m | int | 0-3 | 善良印记数量 |
| evil_marks | E_m | int | 0-3 | 邪恶印记数量 |
| conflict_resolution | C_r | int | -3 to 3 | 冲突解决结果 |

**输出范围：** -3到3（正常游戏范围）
**示例：** 2个善良印记，1个邪恶印记，结果 = 2 - 1 = 1（善良占优）

### 印记解锁公式

`unlock_threshold = required_marks × unlock_multiplier`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| required_marks | R_m | int | 1-5 | 解锁所需印记数量 |
| unlock_multiplier | U_m | float | 1.0-2.0 | 解锁难度倍率 |
| unlock_threshold | U_t | int | 1-10 | 解锁阈值 |

**输出范围：** 1到10（正常游戏范围）
**示例：** 解锁需要3个印记，难度倍率1.5，阈值 = 3 × 1.5 = 4.5（向上取整为5）

## Edge Cases

- **如果印记数量达到上限（3个）**：无法获得更多同类印记，显示"印记已满"提示。
- **如果善良印记和邪恶印记同时存在**：效果相互抵消，取净效果。
- **如果印记效果导致卡牌效果为负数**：卡牌效果设为0，不产生负面效果。
- **如果印记解锁条件不满足**：卡牌保持锁定状态，显示解锁进度。
- **如果印记在战斗中获得**：印记立即生效，影响当前战斗。
- **如果印记在对话中获得**：印记立即生效，影响当前对话。
- **如果印记在存档中丢失**：从云端同步印记数据，确保数据一致性。
- **如果印记数据损坏**：使用备份数据恢复，确保游戏进度不丢失。

## Dependencies

### 硬依赖（系统无法正常工作）

| 系统 | 依赖原因 |
|------|----------|
| **无** | 故事印记系统没有硬依赖 |

### 软依赖（系统可以工作但功能受限）

| 系统 | 依赖原因 |
|------|----------|
| **选择系统** | 印记需要选择系统提供选择数据 |
| **对话系统** | 印记需要对话系统提供对话选择 |

### 被依赖（其他系统依赖本系统）

| 系统 | 依赖原因 |
|------|----------|
| **卡牌数据库** | 卡牌效果需要印记系统提供效果改变 |
| **卡牌战斗系统** | 战斗规则需要印记系统提供规则改变 |
| **UI系统** | 印记显示需要UI系统支持 |

## Visual/Audio Requirements

### 视觉效果
- **印记图标**：每个印记类型有独特的图标（善良：金色光环、邪恶：紫色暗影、中立：灰色平衡、隐藏：闪烁星光）
- **印记获得动画**：玩家做出选择时，印记从选择中浮现，飞入角色状态栏
- **印记效果动画**：印记影响卡牌效果时，卡牌上有印记图标闪烁
- **印记冲突动画**：善良印记与邪恶印记冲突时，两种颜色碰撞消散
- **印记解锁动画**：印记解锁隐藏卡牌时，卡牌从虚空中浮现

### 音频效果
- **印记获得音效**：玩家做出选择时，播放印记获得音效（根据印记类型变化）
- **印记效果音效**：印记影响卡牌效果时，播放印记效果音效
- **印记冲突音效**：善良印记与邪恶印记冲突时，播放冲突音效
- **印记解锁音效**：印记解锁隐藏卡牌时，播放解锁音效

## UI Requirements

### 印记状态显示
- **位置**：角色状态栏，生命值下方
- **内容**：印记图标、印记名称、印记数量
- **大小**：每个印记图标30x30像素
- **动画**：印记获得时，图标从选择中浮现

### 印记详情界面
- **触发方式**：点击印记图标
- **内容**：印记名称、印记类型、印记效果、印记数量、印记来源
- **布局**：列表形式，每个印记一行
- **交互**：点击印记查看详细信息

### 印记效果提示
- **位置**：卡牌详情界面，卡牌效果下方
- **内容**：印记对卡牌效果的影响
- **动画**：印记影响卡牌效果时，提示闪烁

### 印记冲突提示
- **位置**：战斗界面，回合开始时
- **内容**：印记冲突抵消提示
- **动画**：印记冲突时，提示从屏幕中央浮现
