## Overview

**召唤物系统**是战斗系统的核心组件，负责管理召唤类卡牌召唤出的生物/单位。它定义了召唤物的属性、行为、AI和生命周期，为卡牌战斗系统提供召唤物支持。玩家通过召唤物系统感受到"我召唤了一个强大的盟友来帮助我战斗"——召唤物可以攻击敌人、承受伤害、提供增益效果。

召唤物系统读取卡牌数据库的召唤物定义，结合伤害计算系统和状态效果系统，实现召唤物的战斗行为。玩家通过召唤物系统感受到"我的召唤物在战场上为我战斗，我的策略组合产生了强大的连锁反应"。

## Player Fantasy

**召唤物系统**是玩家直接体验的系统——玩家通过召唤卡牌召唤召唤物，召唤物在战场上为玩家战斗。

**情感目标**："我召唤了一个强大的盟友来帮助我战斗"

**玩家应该感受到**：
- **掌控感**：我召唤的召唤物在战场上为我战斗
- **策略深度**：我需要选择合适的召唤物来应对不同的敌人
- **爽快感**：看到召唤物击败敌人时的满足感

**情感锚点**：像游戏王中召唤强力怪兽的爽快感——"我召唤了青眼白龙，这一击致命！"

## Detailed Design

### Core Rules

1. **召唤物属性**：
   - **生命值（HP）**：召唤物可以承受的伤害量
   - **攻击力（ATK）**：召唤物造成的伤害量
   - **防御力（DEF）**：召唤物减少的伤害量
   - **速度（SPD）**：召唤物的行动顺序
   - **元素类型**：召唤物的元素属性（火/水/风/土/雷）

2. **召唤物行为**：
   - **攻击**：召唤物对敌人造成伤害
   - **防御**：召唤物承受伤害，保护玩家
   - **特殊技能**：召唤物可以使用特殊技能（如治疗、增益、控制）

3. **召唤物AI**：
   - **自动攻击**：召唤物每回合自动攻击敌人
   - **目标选择**：召唤物优先攻击元素被克制的敌人
   - **技能使用**：召唤物在满足条件时自动使用特殊技能

4. **召唤物生命周期**：
   - **召唤**：使用召唤卡牌召唤召唤物
   - **存在**：召唤物在战场上存在，每回合行动
   - **死亡**：召唤物生命值为0时死亡，从战场移除
   - **消失**：召唤物持续时间结束时消失

5. **召唤物限制**：
   - **场上上限**：最多同时存在3个召唤物
   - **召唤冷却**：同一召唤物每回合只能召唤一次
   - **元素限制**：召唤物元素类型与环境相同时，获得加成

### States and Transitions

| 状态 | 描述 | 触发条件 |
|------|------|----------|
| **召唤中** | 召唤物正在被召唤 | 使用召唤卡牌时进入 |
| **存在中** | 召唤物在战场上存在 | 召唤完成后进入 |
| **行动中** | 召唤物正在执行行动 | 回合开始时进入 |
| **死亡** | 召唤物生命值为0 | 受到致命伤害时进入 |
| **消失** | 召唤物持续时间结束 | 持续时间结束时进入 |

**状态转换规则**：
- 召唤中 → 存在中：召唤完成后自动进入
- 存在中 → 行动中：回合开始时自动进入
- 行动中 → 存在中：行动完成后自动进入
- 存在中 → 死亡：受到致命伤害时进入
- 存在中 → 消失：持续时间结束时进入

## Formulas

### 召唤物伤害公式

`summon_damage = base_atk × (1 + atk_bonus) × element_modifier`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_atk | A_b | int | 1-100 | 召唤物基础攻击力 |
| atk_bonus | A_bonus | float | 0-1.0 | 攻击力加成（来自Buff/装备） |
| element_modifier | E_m | float | 0.5-2.0 | 元素克制/被克制倍率 |
| summon_damage | D_s | int | 1-200 | 召唤物最终伤害 |

**输出范围：** 1到200（正常游戏范围）
**示例：** 基础攻击力50，加成20%，元素克制1.2x，伤害 = 50 × 1.2 × 1.2 = 72

### 召唤物生命值公式

`summon_hp = base_hp × (1 + hp_bonus)`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_hp | H_b | int | 10-500 | 召唤物基础生命值 |
| hp_bonus | H_bonus | float | 0-1.0 | 生命值加成（来自Buff/装备） |
| summon_hp | H_s | int | 10-1000 | 召唤物最终生命值 |

**输出范围：** 10到1000（正常游戏范围）
**示例：** 基础生命值100，加成30%，生命值 = 100 × 1.3 = 130

### 召唤物防御公式

`summon_defense = base_def × (1 + def_bonus)`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_def | D_b | int | 1-50 | 召唤物基础防御力 |
| def_bonus | D_bonus | float | 0-1.0 | 防御力加成（来自Buff/装备） |
| summon_defense | D_s | int | 1-100 | 召唤物最终防御力 |

**输出范围：** 1到100（正常游戏范围）
**示例：** 基础防御力20，加成50%，防御力 = 20 × 1.5 = 30

### 召唤物速度公式

`summon_speed = base_spd × (1 + spd_bonus)`

**变量：**
| 变量 | 符号 | 类型 | 范围 | 描述 |
|------|------|------|------|------|
| base_spd | S_b | int | 1-50 | 召唤物基础速度 |
| spd_bonus | S_bonus | float | 0-1.0 | 速度加成（来自Buff/装备） |
| summon_speed | S_s | int | 1-100 | 召唤物最终速度 |

**输出范围：** 1到100（正常游戏范围）
**示例：** 基础速度30，加成20%，速度 = 30 × 1.2 = 36

## Edge Cases

- **如果召唤物生命值为0**：召唤物死亡，从战场移除。
- **如果召唤物持续时间结束**：召唤物消失，从战场移除。
- **如果召唤物场上已满（3个）**：无法召唤新召唤物，显示"场上已满"提示。
- **如果召唤物被控制（如眩晕）**：召唤物跳过本回合行动。
- **如果召唤物被沉默**：召唤物无法使用特殊技能，只能普通攻击。
- **如果召唤物被冰冻**：召唤物跳过本回合行动，冰冻效果持续1回合。
- **如果召唤物被石化**：召唤物跳过本回合行动，石化效果持续2回合。
- **如果召唤物被魅惑**：召唤物攻击己方单位，持续1回合。

|------|----------|

## Visual/Audio Requirements

### 视觉效果
- **召唤动画**：召唤物从卡牌中浮现，带有元素特效（火焰/水流/风暴/大地/雷电）
- **召唤物外观**：召唤物有独特的像素艺术外观，根据元素类型变化
- **召唤物血条**：召唤物头顶显示血条，血条颜色根据生命值变化（绿→黄→红）
- **召唤物状态图标**：召唤物头顶显示状态效果图标（Buff/Debuff）
- **召唤物攻击动画**：召唤物攻击时，播放攻击动画
- **召唤物死亡动画**：召唤物死亡时，播放死亡动画（破碎/消散/爆炸）
- **召唤物消失动画**：召唤物消失时，播放消失动画（淡出/消散）

### 音频效果
- **召唤音效**：召唤物被召唤时，播放召唤音效（根据元素类型变化）
- **攻击音效**：召唤物攻击时，播放攻击音效（根据元素类型变化）
- **受伤音效**：召唤物受伤时，播放受伤音效
- **死亡音效**：召唤物死亡时，播放死亡音效
- **特殊技能音效**：召唤物使用特殊技能时，播放技能音效
- **环境加成音效**：召唤物获得环境加成时，播放加成音效

## UI Requirements

### 召唤物状态显示
- **位置**：战斗界面右侧，玩家召唤物区域
- **内容**：召唤物头像、名称、生命值、状态效果图标
- **大小**：每个召唤物卡片100x150像素
- **动画**：召唤物召唤时，卡片从卡牌区域飞入召唤物区域

### 召唤物血条
- **位置**：召唤物头顶
- **内容**：当前生命值/最大生命值
- **颜色**：绿色（>60%）、黄色（30%-60%）、红色（<30%）
- **动画**：受伤时，血条闪烁

### 召唤物状态图标
- **位置**：召唤物头顶，血条下方
- **内容**：状态效果图标（Buff/Debuff）
- **大小**：20x20像素
- **动画**：状态效果生效时，图标跳动

### 召唤物行动提示
- **位置**：召唤物头顶，状态图标下方
- **内容**：召唤物行动提示（攻击/防御/技能）
- **动画**：行动开始时，提示从召唤物头顶浮现
