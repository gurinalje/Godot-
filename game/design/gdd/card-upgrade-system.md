# Card Upgrade System

> **Status**: Designed
> **Author**: User + Agents
> **Last Updated**: 2026-06-03
> **Implements Pillar**: Pillar 2 (Satisfying Depth), Pillar 4 (Persistent Growth)

## Overview

卡牌升级系统是《命运卡牌局》的卡牌成长层，负责管理卡牌的强化和进化。系统定义卡牌的升级路径、升级材料和升级效果，让玩家的卡牌收藏不断成长。

## Player Fantasy

**"我的卡牌在不断变强"**

玩家应该感受到：
- **成长感**：卡牌在不断升级
- **成就感**：升级卡牌带来实力提升
- **策略性**：选择哪些卡牌优先升级
- **投入感**：为升级投入资源的满足感
- **期待感**：期待卡牌升级后的效果

**参考体验**：皇室战争的卡牌升级 + 炉石传说的卡牌强化 + 杀戮尖塔的卡牌升级

## Detailed Design

### Core Rules

**升级规则**：
- **升级等级**：每张卡牌最高可升级到5级
- **升级材料**：需要消耗金币和相同卡牌
- **升级效果**：提升卡牌属性（伤害/效果/持续时间）
- **升级失败**：升级不会失败，但材料消耗递增

**升级路径**：
- **等级1→2**：消耗100金币 + 1张相同卡牌
- **等级2→3**：消耗300金币 + 2张相同卡牌
- **等级3→4**：消耗600金币 + 3张相同卡牌
- **等级4→5**：消耗1000金币 + 5张相同卡牌

**升级效果**：
- **伤害类卡牌**：伤害提升20%/级
- **效果类卡牌**：效果持续时间+1回合/级
- **召唤类卡牌**：召唤物属性提升15%/级
- **环境类卡牌**：环境效果范围+1格/级

**升级限制**：
- 卡牌必须在收藏中（不能在卡组中升级）
- 升级消耗的卡牌会被销毁
- 升级不可撤销

### States and Transitions

| 当前状态 | 触发条件 | 目标状态 | 说明 |
|----------|----------|----------|------|
| 未升级 | 满足升级条件 | 升级中 | 开始升级 |
| 升级中 | 升级完成 | 已升级 | 升级成功 |
| 已升级 | 满足升级条件 | 升级中 | 继续升级 |
| 已满级 | 无法升级 | 已满级 | 达到最高等级 |
| 升级中 | 材料不足 | 未升级 | 升级失败（材料不足） |

## Formulas

**升级消耗公式**：

`upgrade_cost = base_cost × (current_level ^ 1.5)`

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| base_cost | - | int | 100 | 基础消耗 |
| current_level | - | int | 1-4 | 当前等级 |

**输出范围**：100到800
**示例**：base_cost=100，current_level=3 → cost=100×5.2=520（四舍五入）

**升级材料公式**：

`upgrade_materials = current_level`

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| current_level | - | int | 1-4 | 当前等级 |

**输出范围**：1到4
**示例**：current_level=2 → 需要2张相同卡牌

**升级效果公式**：

`upgrade_effect = base_effect × (1 + current_level × 0.2)`

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| base_effect | - | float | 0-100 | 基础效果 |
| current_level | - | int | 1-5 | 当前等级 |

**输出范围**：1.0到2.0倍基础效果
**示例**：base_effect=50，current_level=3 → effect=50×1.6=80

## Edge Cases

| # | 边界情况 | 处理方式 |
|---|----------|----------|
| 1 | **材料不足** | 提示"材料不足"，阻止升级 |
| 2 | **金币不足** | 提示"金币不足"，阻止升级 |
| 3 | **卡牌已满级** | 提示"已满级"，阻止升级 |
| 4 | **卡牌在卡组中** | 提示"请先从卡组移除"，阻止升级 |
| 5 | **升级消耗计算溢出** | 验证计算结果合法性 |
| 6 | **升级效果异常** | 验证效果值合法性 |
| 7 | **卡牌被删除** | 取消升级，退还材料 |
| 8 | **升级中断** | 保存升级进度，允许继续 |

## Tuning Knobs

| 调优参数 | 默认值 | 安全范围 | 极端行为 |
|----------|--------|----------|----------|
| 最大等级 | 5 | 3-10 | 太低=成长受限，太高=过于复杂 |
| 基础消耗 | 100金币 | 50-200 | 太低=太容易，太高=过于困难 |
| 消耗增长指数 | 1.5 | 1.2-2.0 | 太慢=无挑战，太快=过于困难 |
| 效果提升比例 | 20% | 10-30% | 太低=无感，太高=过于强大 |
| 材料需求增长 | 1 | 1-2 | 太慢=太容易，太快=过于困难 |
| 升级成功率 | 100% | 80-100% | 太低=挫败感，太高=无挑战 |
| 升级动画时间 | 2秒 | 1-3秒 | 太短=无仪式感，太长=拖沓 |
| 升级音效 | 有 | 有/无 | 影响升级体验 |

## Visual/Audio Requirements

**视觉效果**：
- **升级界面**：卡牌展示 + 升级预览
- **升级动画**：卡牌发光 + 等级提升动画
- **材料消耗**：材料飞入动画
- **效果提升**：属性数字弹出
- **满级提示**：满级特效 + 成就显示
- **升级失败**：失败提示 + 材料退还动画

**音频效果**：
- **升级开始**：升级开始音效
- **升级成功**：升级成功音效 + 成就音乐
- **升级失败**：升级失败音效
- **材料消耗**：材料消耗音效
- **满级达成**：满级达成音效 + 特殊音乐

## UI Requirements

**卡牌升级界面**：
```
┌─────────────────────────────────────────┐
│  卡牌名称 + 当前等级 + 最高等级         │
├─────────────────────────────────────────┤
│                                         │
│  卡牌展示（卡牌立绘 + 属性）            │
│                                         │
├─────────────────────────────────────────┤
│  升级预览（升级后属性变化）             │
│  伤害：50 → 60 (+10)                    │
│  效果：2回合 → 3回合 (+1)              │
├─────────────────────────────────────────┤
│  升级消耗（金币 + 材料）                │
│  金币：300                              │
│  材料：相同卡牌 ×2                      │
├─────────────────────────────────────────┤
│  操作按钮（升级/取消）                  │
└─────────────────────────────────────────┘
```

**交互规则**：
- 点击升级按钮 → 执行升级
- 点击取消按钮 → 取消升级
- 点击卡牌 → 查看卡牌详情
- 拖拽材料 → 添加升级材料
