# 像素艺术规范文档

## 概述

本文档定义了《命运卡牌局》游戏中所有像素艺术图片的规范和生成提示词。
所有图片应为高质量像素艺术，具有精致的细节和光影效果。

## 通用规范

- **分辨率**: 64x64 或 128x128 像素（角色），32x32 像素（装饰物）
- **风格**: 复古像素艺术，现代光影效果
- **调色板**: 暗色调为主，带有发光效果
- **格式**: PNG，透明背景

## 生成工具推荐

### 1. Midjourney
- 使用 `/imagine` 命令
- 添加参数 `--style pixel` 或 `--style retro`
- 使用 `--ar 1:1` 保持正方形比例

### 2. DALL-E 3
- 直接使用提示词
- 添加 "pixel art" 和 "retro game" 关键词
- 指定尺寸和风格

### 3. Stable Diffusion
- 使用 pixel art 相关的 LoRA 模型
- 添加负面提示词：`blurry, low quality, 3d render, realistic`
- 使用 img2img 进行精细调整

### 4. Aseprite (手动绘制)
- 专业像素艺术软件
- 支持动画制作
- 可以精确控制每个像素
